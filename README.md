# clasePracticandoGitHub
requerimientos funcionales: el sistema debe tener lenguaje natural
                            el sistema debe tener la capacidad de operar en distintos navegadores
                            el sistema tiene que contar con una conexion a la base datos
                            el sistema debe tener la capacidad de procesar pedidos de cualquier cosa
                            el sistema debe saber identificar si le habla un humano o no

requerimientos no funcionales: el login no debe demorarse 
                               debe estar en funcionamiento 24/7
                               manejar muchos usuarios sin tener que fallar 
                               interfaz facil de manejar
